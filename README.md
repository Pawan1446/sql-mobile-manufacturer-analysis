# Mobile Manufacturer Sales Analysis using SQL

## About This Project
This project is based on a mobile sales database designed in a data warehouse format using fact and dimension tables. The objective was to analyze mobile brand performance, model sales, and customer spending trends.

## Tools Used
- Advanced SQL

## What I Did in This Project
- Worked with Fact and Dimension tables
- Found top mobile models year-wise
- Analyzed manufacturer-wise sales performance
- Compared sales for different years
- Identified high-value customers
- Used window functions to rank data
- Applied CTEs for structured queries

## Skills Used
- CTE (WITH clause)
- Window Functions (DENSE_RANK, LAG)
- Year-wise sales comparison
- Ranking and performance analysis
- Data warehouse concepts
